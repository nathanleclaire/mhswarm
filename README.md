## Script for Cross Host Networking

To follow along with the article you can use this script.

Make sure `DIGITALOCEAN_ACCESS_TOKEN` environment variable is set if you're
creating.

To create the hosts:

```console
$ ./ctl.sh up
```

To provision them:

```console
$ ./ctl.sh provision
```

To remove them:

```console
$ ./ctl.sh down
```
